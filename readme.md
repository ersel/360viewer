# 360 web viewer

You can use this with any equirectangular image produced by a 360 camera.

```
yarn
yarn start
```

and then go to http://localhost:3000#URL_OF_IMAGE

Should work cross domains as well.

- Mobile gyroscope support not implemented yet.
